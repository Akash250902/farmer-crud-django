About Farmer Management System

The Farmer Management System is a Django-based web application designed to help manage farmer records efficiently. It allows users to add, view, update, and delete farmer details such as name, contact, crop, and quantity.

This project was created to simplify data handling for agricultural organizations, cooperatives, and NGOs working with farmers.

Technologies Used: Django, Python, Bootstrap, SQLite
